# Public User Manual / 公开版用户手册

## Scope / 范围

The public release is intended for artifact demonstration, teaching, preliminary internal discussion, and preparation for expert review. It does not contain real enterprise responses or operational records.

公开版用于人工制品演示、教学、内部初步讨论和专家评审准备，不包含真实企业答卷或生产运行记录。

## Workflow / 使用流程

1. Open `index.html`.
2. Select Chinese or English.
3. Enter a non-sensitive company profile. Do not enter trade secrets, formulas, personal identifiers, precise customer information, or exact addresses.
4. Complete or import questionnaire responses.
5. Run the analysis to review SPSS-style tables, risk matrix, KPI targets, ROI scenario estimates, and the PDCA improvement workflow.
6. Export browser-generated results only after confirming the data is safe to store or share.

## Interpretation / 结果解读

- Maturity scores indicate diagnostic signals, not certified capability levels.
- Risk scores support prioritisation and do not replace safety, cybersecurity, or compliance assessments.
- ROI values are scenario estimates and not verified financial returns.
- Synthetic examples are for reference only.
