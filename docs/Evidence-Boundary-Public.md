# Evidence Boundary Statement / 证据边界说明

The PCIO Toolkit public release is an artifact demonstration package. It supports design review, educational use, and preparation for expert evaluation.

PCIO Toolkit 公开版是人工制品演示包，适用于设计审查、教学展示和专家评审准备。

## Supported claims / 可支持的表述

- The website and Streamlit package can demonstrate the intended diagnostic workflow.
- Synthetic examples can illustrate module connectivity and output structure.
- The PCIO logic can be inspected as a design artifact.

## Unsupported claims / 不支持的表述

- No causal claim is made about productivity, quality, energy, safety, or financial improvement.
- No real enterprise raw data is published.
- No original enterprise case document or expert raw evaluation record is included.
- ROI outputs are scenario estimates only, not verified real returns.

如需学术合作或专家审核材料，请联系作者。该材料不在公开版本中提供。
