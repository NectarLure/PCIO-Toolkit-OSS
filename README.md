# PCIO Toolkit Public Release / PCIO Toolkit 公开发布版

This directory is a public website release of the PCIO Toolkit. It preserves the current static website layout and browser-side demonstration while excluding private research drafts, raw enterprise data, original case documents, and expert raw evaluation records.

本目录是 PCIO Toolkit 的公开网站发布版。它保留当前静态网站的页面结构和浏览器端演示功能，同时排除论文草稿、真实企业原始数据、企业案例原文和专家原始评价记录。

## Included / 已包含

- Static website: `index.html`, `paper-capture.html`, `assets/`, `downloads/`.
- Public documentation: `docs/`.
- Synthetic examples only: `examples_synthetic/`.
- Blank expert review forms: `downloads/expert_review_form.*`.
- License and citation metadata: `LICENSE`, `CITATION.cff`.

## Not Included / 未包含

Private manuscript materials, real survey data, raw T0 files, enterprise case originals, employee information, equipment identifiers, customer/supplier details, addresses, and sensitive tank-area records are not included. ROI outputs are scenario estimates only and must not be interpreted as verified financial returns.

本公开版不包含论文私有材料、真实问卷数据、T0 原始文件、企业案例原文、员工信息、设备编号、客户/供应商细节、地址或罐区敏感记录。ROI 输出仅为情景估算，不代表真实收益验证。

## Use / 使用

Open `index.html` in a browser or deploy the whole `public_release/` directory to GitHub Pages, Vercel, Netlify, or any static web server. All page links are relative to this directory.

可直接用浏览器打开 `index.html`，也可将整个 `public_release/` 部署到 GitHub Pages、Vercel、Netlify 或任意静态服务器。所有页面引用均为目录内部相对路径。
